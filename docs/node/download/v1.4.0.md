# Release Notes for Klaytn v1.4.0 <a id="release-notes-for-klaytn-v1-4-0"></a>

## 새로운 기능 <a id="new-features"></a>
- 키스토어 V4
  - Klaytn 계정 활용을 돕는 새로운 키스토어 표준  [KIP-3](https://klaytn.github.io/kips/KIPs/kip-3)을 도입했습니다.

## 수정사항 <a id='fixes'></a>
- Fixed intermittent bad block occurrence caused by timing issue when updating `stakingInfo`.
- Fixed panic during execution of `debug.traceBlock` because of the timing issue between the API and garbage collection of trie nodes.
- Fixed a bug report during block synchronization.

## 개선사항 <a id='improvements'></a>
- 최적화된 스토리지 트리
  - Klaytn은 이제 모든 트랜잭션이 아니라 모든 블록마다 스토리지 루트 해시를 계산합니다.
  - Klaytn은 트리 해시 루트를 동시에 계산합니다.
- 증가된 상태 트리 캐시 크기
  - 기본 캐시 크기 증가로 향상된 성능
- Badger DB 버전 v1.6.0으로 업그레이드.
- 매트릭스 패키지 교체
- 업데이트된 패키지 설명
- 더 넓어진 테스트 코드 커버리지
