# Release Notes <a id="release-notes"></a>

이 버전은 프로토콜 업그레이드 수정사항을 포함하고 있습니다. Baobab 테스트 네트워크에서 블록넘버 #80295291 부터 적용됩니다. 모든 Baobab 테스트 노드는 해당 블록넘버 이전에 반드시 업그레이드해야 합니다.

## Baobab 네트워크 프로토콜 업그레이드
- 이더리움 런던 EVM 호환성을 위한 BaseFee EVM opcode 도입 (#1065, #1066, #1096)
- Baobab 네트워크에 대한 런던 EVM 하드포크 넘버 설정 (#1080)

## New Features
- PN 노드에 스팸 조절기 도입 (#1092)
- 외부 노드로 부터 오는 트랜잭션을 거부할 수 있는 트랜잭션 풀 플래그 추가 (#1075)
- 하드포크 블록 넘버를 셋업하는 툴 업데이트 (#1070, #1082)

## Improvements
- solc v0.8 아웃풋 포맷 지원 (#1030)

## Fixes
- darwin/amd64용 go v1.17 버전을 지원하기 위한 syscall 라이브러리 업데이트 (#1048)
- 노드 재시작 시 로컬 트랜잭션 삭제 방지 (#1084)
- Epoch 주기 블록의 거버넌스 데이터 업데이트 지연이 되는 문제를 수정 (#1089)

## 기타
- API에서 pending block 지원하도록 변경 (#1064)
